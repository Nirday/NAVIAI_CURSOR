# Visual Design Themes for Website Templates

## Overview

When SMB owners choose a website template, they're selecting a **visual design style** that defines how their website will look and feel. All templates are:
- ✅ **SEO-Friendly**: Built with semantic HTML, proper heading structure, Schema.org markup
- ✅ **Easily Editable**: Simple drag-and-drop interface, clear content blocks
- ✅ **API Integration Ready**: Support for embeds, widgets, and third-party integrations

## 8 Visual Design Themes

- Clean lines, ample white space
