# Website Template Theme Strategy

## Theme Design Philosophy

Each template uses a carefully crafted theme that includes:
1. **Color Palette** - 6 color roles for consistent branding
2. **Typography** - Font pairings optimized for readability and brand voice
3. **Layout Style** - Visual structure (centered, full-width, grid, etc.)
4. **Section Order** - Recommended content flow

## Color Palette Structure

Every theme includes these 6 color roles:

### Primary Color
- **Purpose**: Main brand color, CTAs, links, key elements
- **Usage**: Buttons, headers, important highlights
- **Examples**: Blue (#2563EB) for professional, Red (#DC2626) for bold

### Secondary Color
- **Purpose**: Supporting brand color, depth
- **Usage**: Hover states, secondary buttons, accents
- **Examples**: Darker shade of primary

### Accent Color
- **Purpose**: Highlights, callouts, special elements
- **Usage**: Badges, notifications, special CTAs
- **Examples**: Gold for luxury, Orange for energy

### Background Color
- **Purpose**: Main page background
- **Usage**: Body background, clean canvas
- **Examples**: White (#FFFFFF), Cream (#FFF8F0) for food

### Surface Color
- **Purpose**: Cards, sections, elevated elements
- **Usage**: Content cards, feature boxes, form backgrounds
- **Examples**: Light gray (#F8FAFC), White (#FFFFFF)

### Text Color
- **Purpose**: Primary text content
- **Usage**: Headings, body text, readable content
- **Examples**: Dark gray (#1F2937), Black (#000000)

## Typography Strategy

### Font Pairing Principles

1. **Contrast**: Heading font should contrast with body font
2. **Readability**: Body font must be highly readable
3. **Brand Voice**: Fonts should match template personality

### Font Categories Used

**Heading Fonts** (Display/Serif/Sans-Serif):
- Inter, Roboto, Poppins (Modern/Sans)
- Playfair Display, Crimson Text (Elegant/Serif)
- Montserrat, Oswald (Bold/Sans)
- Raleway, Nunito (Creative/Sans)

**Body Fonts** (Readable):
- Inter, Roboto, Open Sans (Clean/Readable)
- Lora, Source Sans Pro (Elegant/Readable)
- Lato (Versatile)

## Color Psychology Applied

- **Blue**: Trust, professionalism, stability (Professional, Medical, Real Estate)
- **Red**: Energy, urgency, passion (Food, Fitness, Bold)
- **Green**: Growth, health, trust (Trustworthy Business, Wellness)
- **Purple**: Creativity, luxury, wisdom (Creative, Education)
- **Gold/Yellow**: Luxury, premium, warmth (Elegance, E-commerce)
- **Black/Gray**: Sophistication, minimalism, professionalism (Elegance, Minimal)
- **Orange**: Energy, enthusiasm, creativity (Bold, Food, Fitness)

## Layout Styles

1. **Centered**: Content centered with max-width (Professional, Elegant)
2. **Full-Width**: Edge-to-edge content (Bold, Food, Fitness)
3. **Grid**: Card-based grid layout (Creative, E-commerce, Real Estate)
4. **Sidebar**: Traditional sidebar layout (not currently used, but available)

## Best Practices

1. **Contrast Ratios**: All text meets WCAG AA standards (4.5:1 minimum)
2. **Accessibility**: Color-blind friendly palettes
3. **Brand Consistency**: Colors work together harmoniously
4. **Readability**: Body fonts are optimized for screen reading
5. **Mobile-First**: All themes work on mobile devices

## Customization

Users can:
- Override any template color
- Change fonts
- Mix template elements
- Create fully custom themes
